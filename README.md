### Описание проекта
В компании Booking хотят обзавестить моделью предскахания рейтинга отеля.

### Какой кейс решаем?
Одна из проблем компании — это нечестные отели, которые накручивают себе рейтинг.

### Условие для обработки данных
Условия зады в описании каждой задачи, для оформления решения не использованы готовые шаблоны. Решение оформлено полснотью в самостоятельно разработанном ноутбуке.

### Метрика качества
Результаты обработаки данных оценивались выведенными проверочными параметрами, заданными в системе, в том числе метрикой Mean Absolute Percentage Error (MAPE), которая показывает среднюю абсолютную процентную ошибку предсказанных значений от фактических.

### Рузультат

В качестве резельтатов преобразованы данные для удобства и правильности работы с ними, получены количесвтенные и качественные результаты, сформулированы выводы для построения модели обучения в дальнейшем.
