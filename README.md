### Описание проекта
Компания HeadHunter хочет построить модель машинного обучения, которая будет рекомендовать вакансии клиентам агентства, претендующим на позицию Data Scientist.

### Какой кейс решаем?
Необходимо понять, что из себя представляют данные и насколько они соответствуют целям проекта и затем провести анализ вакансий.

### Условие для обработки данных
Условия зады в описании каждой задачи, для оформления решения не использованы готовые шаблоны. Решение оформлено полснотью в самостоятельно разработанном ноутбуке.

### Метрика качества
Результаты обработаки данных оценивались выведенными проверочными параметрами, заданными в системы. При некорреткном выводе результов они не принимались системой

### Рузультат

В качестве резельтатов преобразованы данные для удобства и правильности работы с ними, получены количесвтенные и качественные результаты, сформулированы выводы для построения модели обучения в дальнейшем.
