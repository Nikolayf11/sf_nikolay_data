### Описание проекта
Компания HeadHunter хочет построить модель машинного обучения, которая будет рекомендовать вакансии клиентам агентства, претендующим на позицию Data Scientist.

### Какой кейс решаем?
Необходимо понять, что из себя представляют данные и насколько они соответствуют целям проекта и затем провести анализ вакансий.

### Условие для обработки данных
Условия задыны в описании каждой задачи, для оформления решения не использованы готовые шаблоны. Решение оформлено полснотью в самостоятельно разработанном ноутбуке.

### Метрика качества
Результаты обработаки данных оценивались выведенными проверочными параметрами, заданными в системе (https://apps.skillfactory.ru/learning/course/course-v1:SkillFactory+DSPRMGU+2023_FEB/block-v1:SkillFactory+DSPRMGU+2023_FEB+type@sequential+block@add6a0ff487e40dcb8b9c13dbab2840a/block-v1:SkillFactory+DSPRMGU+2023_FEB+type@vertical+block@43d3d18135474c41897d3108dd10b492). При некорреткном выводе результов они не принимались системой.

### Рузультат
Получены количесвтенные и качественные результаты анализа вакансий, влючая анализ зарплат, регионов, требований к графику работы и занятости, сформулированы выводы для построения модели обучения в дальнейшем.
